# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅         |
| Older   | ❌         |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through a **GitLab**'s [issue](https://gitlab.com/kilianpaquier/website/issues)
and make sure to tick checkbox `Turn on confidentiality: Limit visibility to project members with at least the Planner role.`.
